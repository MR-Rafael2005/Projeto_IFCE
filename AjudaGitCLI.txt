git init [Iniciar repositorio]
git rev-parse --show-toplevel [Ver onde esta a pasta .git que contem o repositorio]
git status [Ver a stuação atual do repositio e branch]
git add * [Adiciona todos os arquivos ao repositorio]
git commit -m "" [Salva as alterações em um commit e entre as aspas se coloca uma mensagem tipo "Pagina de login finalizada"]
git log --oneline [Mostra um historico de commits e alterações no repositorio/branch]
git checkout [Permite navegar entre commits e branches]
git revert [Remove todas as alterações de um commit]
git reset [Volta para um commit especifico PERMANENTEMENTE!!!]
git branch [Verifica em qual branch se está]
git branch <novo-branch> [Quando se coloca algo no lugar de <novo-branch> se cria um novo branch, tipo: git branch sistema_agendamento]
git merge <nome_branch> [Juntar branches, puxando um branch para o branch atual, tipo: git merge master(Supondo que se esta no branch sistema_agendamento por exemplo, ou em outro)]
[OBS: Pode ocorrer de se abrir o VIM, para fechar digite :q! e aperte ENTER]
git remote add origin [Adiciona uma seed/link para o repositorio no github, ja vem definida automaticamente quando o repositorio é clonado]
git push [Envia as alterações ate o ultimo commit para o repositorio no github]
git pull [Atualiza o repositorio local com o repositorio do github]
